
#   JavaScript Basic Math & Logic Questions


## ✅ 1. Check if a Number is Divisible by 3 and 5

    Question:
        Write a function that checks whether a number is divisible by both 3 and 5.

## ✅ 2. Find the Factorial of a Number

Question:
Create a function that takes a number and returns its factorial using a loop.
Example: factorial of 4 → 24.

## ✅ 3. Check if a Number is Even or Odd

    Question:
        Write a function that checks whether a number is even or odd and prints the result.

## ✅ 4. Print Numbers from 1 to N

    Question:
        Write a function that takes a number n and prints numbers from 1 to n using a loop.

## ✅ 5. Find the Square of a Number

    Question:
        Write a function that returns the square of a given number.
Example: 5 → 25.

## ✅ 6. Sum of All Even Numbers in an Array

    Question:
        Given an array of numbers, write a function that calculates the sum of all even numbers inside the array.

## ✅ 7. Sum of First N Natural Numbers

    Question:
        Write a function that returns the sum of the first N natural numbers.
Example: N = 10 → 55.

## ✅ 8. Sum of All Odd Numbers in an Array

    Question:
        Given an array, create a function that finds the sum of all odd numbers present in the array.
